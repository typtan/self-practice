1. การสร้าง Array และ loop แสดงค่า Array ว่ามีอะไรบ้าง มีกี่ตัว

Ans หนูได้ลองสร้าง Array โดยการใช้ let เพราะไม่มีการที่จะต้องแก้ไขข้อมูล 
แล้วหนูก็ได้ลองใช้ for loop เพื่อวนดูค่าใน array ว่ามีอะไรบ้าง มีกี่ตัว โดยจะกำหนดค่า i = 0 ไว้ก่อน
ไว้เช็คว่าถ้า i ยังน้อยกว่าจำนวนตัวใน array ให้วน loop ต่อ โดยแต่ละ loop จะมีการบวกค่า i เพ่ิมทีละ 1 เพื่อเช็คว่า loop กี่ตัวแล้ว
ยังน้อยกว่าจำนวนตัวใน array รึเปล่า ถ้ายังน้อยกว่าอยู่ก็ loop ไปเรื่อยๆ และแสดงค่ากับจำนวนใน array 

2. ทำโจทย์การรวม array ด้วย spread operator

Ans หนูลองใช้ spread operator เนื่องจากการสอบในปีหนึ่งหนูมีการทำข้อสอบตรงส่วนนี้พลาดเลยอยากกลับมาลองทำอีกรอบ
หนูสร้าง array A และ array B และสร้าง array C มาเพื่อใช้ spread operator นำค่าทั้งหมดของทั้ง 2 array มารวมกันเป็น array C

3. โจทย์การทำเมนูร้านอาหาร

Ans หนูได้นำความรู้ทั้งหมดในการเรียน week01 มารวมแล้วทำเป็นระบบการดูเมนู โดยมีการสร้าง array อาหารและน้ำ
และให้แสดงค่าออกมาเป็น menu แต่ละหมวด (อาหาร, น้ำ) และแสดงจำนวนของเมนูแต่ละหมวด และในการประกาศตัวแปร
เมนูหนุได้ใช้ const ในการประกาศเลยทำให้สามารถเปลี่ยนค่าใน array เมนูได้ ได้ลองทำทั้ง แก้ไขเมนูคือใส่ค่าทับไปเลยด้วยการบอก index
และเพิ่มเมนูอาหารด้วย .push เข้าไปหลังสุดใน array เมนูอาหาร และได้ลบเมนูน้ำเมนูแรกโดยใช้ delete และเจาะจง index 
และเพิ่มเมนูน้ำด้วย .push เข้าไปหลังสุดใน array เมนูน้ำ พอแก้ไขเมนูเสร็จทุกอย่างก็ log ออกมาดูเมนูใหม่
