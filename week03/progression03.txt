
1. เป็นการเขียน function นับจำนวนครั้งของคำแต่ละคำที่มี โดยเช็คค่าก่อนว่าเป็น null หรือ undefined ไหม
ถ้าเป็นให้ return undefined แล้วแปลงเป็นตัวพิมพ์เล็กทั้งหมดเพื่อให้เทียบค่าได้และแยกคำออก แล้วก็สร้าง object มาเก็บค่า
โดยการเช็คคำคือการเช็คว่ามีคำนั้นเป็น key ใน object แล้วยัง ถ้ามีแล้วให้ + ค่าเพิ่มไป 1 แต่ถ้าไม่เคยมีให้สร้าง
key ใหม่เป็น 1
